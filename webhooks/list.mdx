# List webhooks

Returns all webhooks configured for your account.

## Endpoint

```http
GET /v1/webhooks
Host: api.driftguard.dev

Example
bash
curl -s https://api.driftguard.dev/v1/webhooks \
  -H "Authorization: Bearer $DRIFTGUARD_KEY"
