# Get usage

Returns your current plan, credits, and rate limits.

## Endpoint

```http
GET /v1/usage
Host: api.driftguard.dev

Example
bash
curl -s https://api.driftguard.dev/v1/usage \
  -H "Authorization: Bearer $DRIFTGUARD_KEY"
json
{
  "plan": "starter",
  "credits_total": 500,
  "credits_used": 23,
  "credits_remaining": 477,
  "period_start": "2026-02-01T00:00:00Z",
  "period_end": "2026-02-29T23:59:59Z",
  "rate_limit_per_minute": 60
}

Use this endpoint to build dashboards or alerts around credit usage.
