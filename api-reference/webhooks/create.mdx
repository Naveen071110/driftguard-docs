# Create Webhook

Register a webhook endpoint that DriftGuard will call whenever a check completes with certain statuses.

## Endpoint

<div className="api-endpoint">

```http
POST /v1/webhooks
Host: api.driftguard.dev
```

</div>

## Request

```json
{
  "name": "Slack data-alerts channel",
  "target_url": "https://hooks.slack.com/services/...",
  "event_types": ["BREAKING", "WARNING"],
  "contract_id": null
}
```

| Field       | Type          | Required | Description                                |
| ----------- | ------------- | -------- | ------------------------------------------ |
| name        | string        | ❌       | Friendly label for the webhook             |
| target_url  | string        | ✅       | URL DriftGuard will POST to                |
| event_types | array         | ✅       | Statuses that trigger this webhook         |
| contract_id | string / null | ❌       | Scope to a single contract or null for all |

## Response

```json
{
  "webhook_id": "wh_01HX8ABCD",
  "name": "Slack data-alerts channel",
  "target_url": "https://hooks.slack.com/services/...",
  "event_types": ["BREAKING", "WARNING"],
  "contract_id": null,
  "created_at": "2026-02-27T19:22:00Z",
  "active": true
}
```
