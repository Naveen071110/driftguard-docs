# List contracts

Returns all contracts for the authenticated account.

## Endpoint

```http
GET /v1/contracts
Host: api.driftguard.dev

Query parameters
Name	Type	Description
limit	integer	Max records to return (default 50, max 100)
cursor	string	Pagination cursor from previous response

Example
bash
curl -s "https://api.driftguard.dev/v1/contracts?limit=20" \
  -H "Authorization: Bearer $DRIFTGUARD_KEY"
json
{
  "data": [
    {
      "contract_id": "ctr_01HX7KMQZP",
      "name": "users_production",
      "version": 1
    },
    {
      "contract_id": "ctr_01HX7R5ABC",
      "name": "orders_production",
      "version": 2
    }
  ],
  "next_cursor": null
}
